## 在 TCP/IP 模型中，负责端到端可靠传输的是哪一层？

- [ ] 网络层
  网络层（IP）负责寻址与路由，不保证可靠交付。
- [x] 传输层
  传输层的 TCP 通过确认、重传、滑动窗口实现可靠传输。
- [ ] 数据链路层
  链路层负责相邻节点间的帧传输。
- [ ] 应用层
  应用层面向具体应用协议，如 HTTP、DNS。

> Explain: TCP 工作在传输层，提供面向连接、可靠、有序的字节流服务。

## HTTP 默认使用的传输层协议和端口是？

- [x] TCP，端口 80
  HTTP/1.1 与 HTTP/2 基于 TCP，明文默认端口 80。
- [ ] UDP，端口 80
  HTTP/1.1 不使用 UDP（HTTP/3 才基于 QUIC/UDP）。
- [ ] TCP，端口 443
  443 是 HTTPS 的默认端口。
- [ ] UDP，端口 53
  53 是 DNS 的默认端口。

## （多选）以下哪些协议工作在应用层？

- [x] HTTP
  超文本传输协议，应用层。
- [x] DNS
  域名解析协议，应用层。
- [ ] IP
  网际协议，工作在网络层。
- [ ] TCP
  传输控制协议，工作在传输层。
