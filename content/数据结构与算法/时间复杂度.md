## 下面哪个增长最快（当 $n \to \infty$）？

- [ ] $O(n \log n)$
  线性对数级，介于线性与平方之间。
- [ ] $O(n^2)$
  平方级，比指数级慢得多。
- [x] $O(2^n)$
  指数级，增长远快于任何多项式。
- [ ] $O(n)$
  线性级，增长最慢。

> Explain: 增长速度排序：$O(n) < O(n\log n) < O(n^2) < O(2^n)$。

## 在一个长度为 $n$ 的已排序数组中进行二分查找，时间复杂度是？

- [x] $O(\log n)$
  每次比较都把搜索区间减半。
- [ ] $O(n)$
  这是线性查找的复杂度。
- [ ] $O(n \log n)$
  这是高效比较排序的复杂度。
- [ ] $O(1)$
  只有直接索引访问才是常数级。

## （多选）以下哪些操作在平均情况下是 $O(1)$？

- [x] 哈希表按键查找
  理想哈希下平均常数时间。
- [x] 数组按下标访问
  直接寻址，常数时间。
- [ ] 链表按值查找
  需要遍历，平均 $O(n)$。
- [ ] 在无序数组中查找最大值
  必须扫描全部元素，$O(n)$。
