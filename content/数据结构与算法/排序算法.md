## 下列排序算法中，在最坏情况下时间复杂度依然保持为 $O(n \log_2 n)$，且属于不稳定排序的是哪一个？

- [ ] 归并排序
  归并排序虽然在最坏情况下时间复杂度为 $O(n\log_2 n)$，但它是一种稳定的排序算法。
- [ ] 冒泡排序
  冒泡排序在最坏情况下的时间复杂度为 $O(n^2)$，且它是稳定的。
- [ ] 快速排序
  快速排序在最坏情况下（如序列基本有序时）时间复杂度会退化至 $O(n^2)$。
- [x] 堆排序
  堆排序在最好、最坏和平均情况下的时间复杂度均为 $O(n\log_2 n)$，且由于存在跨距离的元素交换，属于不稳定排序。

> Explain: 判断关键有两点：**最坏情况仍为 $O(n\log_2 n)$** 排除了快速排序与冒泡排序；**不稳定** 排除了归并排序。只有堆排序同时满足。

## 以下关于快速排序的说法，哪一项是正确的？

- [ ] 快速排序是稳定排序。
  快速排序在分区时会交换非相邻元素，因此是不稳定的。
- [x] 快速排序的平均时间复杂度为 $O(n \log_2 n)$。
  平均情况下每次划分较均匀，递归深度约为 $\log_2 n$，每层处理 $O(n)$。
- [ ] 快速排序的空间复杂度恒为 $O(1)$。
  递归调用栈在平均情况下为 $O(\log_2 n)$，最坏为 $O(n)$。
- [ ] 快速排序在任何输入下都优于归并排序。
  对基本有序的输入，朴素快排会退化到 $O(n^2)$，可能慢于归并排序。

## （多选）以下哪些排序算法是**稳定**的？

- [x] 归并排序
  归并时相等元素保持相对次序，稳定。
- [x] 冒泡排序
  仅相邻元素比较交换，相等元素不会交换，稳定。
- [ ] 快速排序
  分区交换非相邻元素，不稳定。
- [ ] 堆排序
  建堆与调整会跨距离交换，不稳定。
